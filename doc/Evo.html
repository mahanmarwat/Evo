Evo 1.0 Documentation

Evo:

Evo is a GUI application for PTCL Evo Wingle. Evo is a very lightweight (less than 100 KB) application which shows network status and statistics. Evo is designed from ground up to support multi-platform and have hooks for future improvements.

(Default Password: admin)

 

Evo Features:

  • Lightweight (less then 100 KB)
  • Support multi-platform. (Unix, Linux, Window, Mac)
  • Beautiful theme
  • Password protection support

Open source world.

Downloading and Installing:

Evo code has been written in Python Programming Language, and its GUI is GTK based.

Therefore, to run Evo, Python Interpreter and PyGi are required.

Follow the below steps, to run Evo:

 

  1. First-of-all download Python 3 from http://www.python.org and install it.
  2. Then download PyGi from https://wiki.gnome.org/Projects/PyGObject and install it.
  3. After that, download Evo.rar.
  4. Then extract the Evo.rar and click on Evo.pyw to run it.

 

* Download the versions of Python Interpreter and PyGi for your OS.

* If you are having any problem in downloading, installing mail me at: evoappdeveloper@gmail.com

FAQ:

Why Evo?

PTCL Evo Wingle does not have its own GUI.

I do not have PTCL Evo Wingle, can I Use Evo with other devices?

Yes, But then Evo will only indicate connection (Green button on toolbar)/disconnection (Red button on toolbar). Evo will not provide other info like statistics etc.

Why the daunting process installing Python, PyGi and Evo?

I am working on it. And very soon release self executable version without installing Python, PyGi.

How to close Evo, when decoration set to False?

Click on Evo application to select it. And press (Alt + F4). This works on window platform. Please check your OS manual for a keyboard shortcut to close an application.
